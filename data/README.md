# Data folder

This folder is used to store input files for the reanalysis.

This reanalysis uses publicly available mTEC scRNA-seq data from:

Wells et al. Combined transient ablation and single-cell RNA-sequencing reveals the development of medullary thymic epithelial cells. eLife, 2020.

GEO accession:

GSE137699

Original GEO dataset title:

Combined transient ablation and single cell RNA sequencing reveals the development of medullary thymic epithelial cells

The metadata file is included in this repository:

* `controls_meta.csv`

The following large processed expression files are **not included** in this repository:

* `GSE137699_allSamples.csv`
* `GSE137699_combinedControl.csv`

Before running the scripts, download or prepare these GEO-derived processed CSV files and place them in this folder using the exact file names listed above.

These files are derived from the public GEO dataset and/or local preprocessing of the GEO files.
