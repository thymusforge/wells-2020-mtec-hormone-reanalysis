# Wells 2020 mTEC Hormone Reanalysis

This repository contains educational reanalysis code for Wells et al. eLife 2020 mTEC scRNA-seq data.

The project focuses on medullary thymic epithelial cell (mTEC) development and explores sex hormone receptor signaling, estrogen receptor signaling, androgen receptor signaling, progesterone receptor-related GO terms, and marker gene expression across Figure 1b-style epithelial cell states.

## Citation

Wells et al. Combined transient ablation and single-cell RNA-sequencing reveals the development of medullary thymic epithelial cells. eLife, 2020.

## Data source

The original RNA-seq data used in Wells et al. eLife 2020 were deposited in NCBI GEO under accession number GSE137699.

Because the processed CSV files are large, this repository does not include the input data files directly. To run the scripts, download or prepare the corresponding GEO-derived files and place them in the `data/` folder with the following names:

- `controls_meta.csv`
- `GSE137699_allSamples.csv`
- `GSE137699_combinedControl.csv`

The `output/` folder is generated automatically by the scripts and is not tracked by Git.

## File Structure

```text
data/
  Raw input CSV files used by the reanalysis.
  These files should not be edited by hand.

scripts/
  Numbered R scripts for inspection, object construction, UMAP reanalysis,
  hormone-related pathway scoring, and gene-level visualization.

output/
  Automatically generated results, figures, tables, and RDS objects.
  This folder is ignored by Git and can be regenerated by rerunning scripts.
```

## Recommended Run Order

Run scripts from the project root, or from any subdirectory inside the project. Each script automatically detects the project root by finding `data/` and `scripts/`.

```r
source("scripts/00_inspect_geo_files.R")
source("scripts/01_create_seurat_object.R")
source("scripts/02_run_pca_umap.R")
source("scripts/03_sex_hormone_programs_figure1b_v2.R")
source("scripts/04_count_kegg_go_estrogen_androgen_terms.R")
source("scripts/04_find_pgr_go_terms.R")
source("scripts/05_go_receptor_signaling_scores_figure1b.R")
source("scripts/05_specific_genes_dotplot_go_receptor_signaling.R")
```

`03_sex_hormone_programs_figure1b_v2.R` is the recommended Figure 1b hormone-program script. The older `03_sex_hormone_programs_figure1b.R` is kept for reference and comparison, but students should use the v2 script unless they have a specific reason to inspect the earlier version.

## Notes for Students

- Do not manually edit files in `data/`.
- Do not manually edit files in `output/`; they are generated by the scripts.
- If an output file is missing, rerun the corresponding numbered script and the scripts before it.
- Some scripts require packages such as `Seurat`, `SeuratObject`, `Matrix`, `ggplot2`, and `msigdbr`.
- `msigdbr` may report that human MSigDB terms are mapped to mouse orthologs. This is expected for scripts that score mouse expression data using MSigDB gene sets.

## Main Outputs

- `output/control_seurat_initial.rds`: initial Seurat object from the control expression matrix.
- `output/control_seurat_pca_umap.rds`: object after PCA, clustering, and UMAP reanalysis.
- `output/03_sex_hormone_figure1b/`: receptor expression and hormone program summaries.
- `output/04_kegg_go_sex_hormone_terms/`: KEGG/GO term searches for estrogen, androgen, and Pgr-related terms.
- `output/05_go_receptor_signaling_scores/`: GO receptor signaling module scores and gene-level dot plots.
